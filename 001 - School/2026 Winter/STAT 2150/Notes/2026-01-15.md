# R Basics
*R* code can be comprised of a variety of things, such as functions, calculations and plots

> Always create your code in an *R* Script File (.R file)

> Comment your code using # to allow others to know what you are doing

When something is created in *R*, it is an object
- These are divided by classes
	- String
	- Numeric
	- Logical
	- Matrix
	- Data Frame
	- Lists
	- etc.
